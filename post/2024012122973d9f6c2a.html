备份固件以便在搞砸之后可以救砖

请在任何刷机操作前务必备份

取决于这些棒子的体质，即使是正确的操作也有出错的可能。所以，无论如何，在任何危险操作前备份！！！

汝也很有可能漏掉一些事情，比如棒子原始系统里的一些信息(IMEI,配置项)，这时只能刷回原系统查看，如果没有备份，即使同一型号的其他棒子也未必拥有一样的内容

工具链

这些工具在互联网上都能找到，酷安水遍的刷机工具箱里也都提供了集成。

注意查毒

miko service tool

通常使用这个工具将整个固件打包为单个bin文件，很多渠道中提到的miko包就是指这个

写在前面，据咱的实际经验，该工具的备份/还原在某些情况下并不可靠，建议同时使用下文的Qualcomm Premium Tool进行备份，以防救砖的时候被气死:(

使用方式

  1. 进入9008模式
    参考后文9008模式章节

  2. 备份
    如图，点击Read -> Partition Backup/Erase，再点击Load Parition Structure读取分区列表

然后，全选分区，点击Read Full Image，弹出文件保存框，选择一个保存路径，确定后开始备份，进度标注在界面下方的进度条。


确认输出的是Success这个备份即完成。

注意：这个备份文件会和设备的emmc一样大，如果备份的是4G的棒子，则会生成4G大小左右的文件。但是该文件作为硬盘镜像，存在大量重复的空数据，所以对其进行压缩是一个好注意，这可以将一个默认的4G安卓固件压缩为300MB左右

  1. 还原

如图，选择Flash -> emmc block0 flasher,然后双击中间的框选择备份文件，然后点击FLASH，开始刷入，进度条在界面下放展示

注意：如果这里没有Success，则表示刷入失败，这时即使能进系统，那大概率也是部分损坏的系统。这里可以多试几次，但如果始终失败，那么汝最好有QPT备份并尝试使用QPT的9006还原进行恢复

Qualcomm Premium Tool

这就是所谓的QPT工具
通常使用这个工具可以打出单个分区的备份包，同时它的两个功能9008 to 9006 和9006 Block 0 Write两个功能可以几乎百分百成功救砖(至少咱没失败过)

常规备份

这将备份每一个分区为单独的文件，以便之后可以单独还原特定分区

注意：这不是全量备份/还原的法子，它确实备份了每一个分区，但是汝不会想一个一个地还原回去的，一个棒子可能会有几十个分区的说

  1. 进入9008模式

  2. 点击Qualcomm -> partition，首先缺点单选框在Scan位置，即扫描分区，然后点击Do Job

  3. 扫描分区后，单选框应自动跳到Backup，即备份位置，这时勾选右侧Backup ALL复选框，然后点击Do Job，这时应弹出文件夹选择框，选择一个干净的文件夹。

    当左下角显示Backup finished的时候，即完成备份

保底救砖

这就是咱前文提到的，使用9008 to 9006 和9006 Block 0 Write两个功能进行救砖

备份
首先进入9008模式，然后选择Qualcomm -> Block 0 -> Block0 Read，点击Read（9008）
这里的路径选择框似乎没有意义，保存路径在下一步中选择

之后在弹出的窗口中拖动滚动条到最底下，并点击最后一个分区，最后点击backup now，这时选择保存路径，然后开始备份

如下图，备份完成

还原
首先进入9008模式，然后如图，选择General -> Reboot Options -> 9008 to 9006，注意复选框一定选择到MSM8916。
注意：如果是酷安水遍的刷机工具箱自带的QPT，这里很有可能被精简掉了，汝可能需要google一个完整版来用
点击go，过一会设备应自动重启，进入9006模式

点击Qualcomm -> Block 0，找到9006 Block 0 Write

  1. 首先点击最右边的Refresh刷新设备列表，并在左侧下拉栏中选择正确的设备
  2. 点击左侧Select Block0功能的文件选择按钮，找到备份的QPT Block0备份包
  3. 点击Write开始刷入

    注意：只有左侧开始这样刷新显示才是正常开始刷入，并可以看到百分比

    如果提示错误，多按几次Write重试几次(或者几十次)，还是不行重新刷新/选择设备，再不行重启QPT
    总之这东西很容易启动报错，而且原因未知，但是总是能重试解决，并且一旦开始读条，咱就没见它失败过

    如上图所示，刷入完成，拔出棒子并重新插入，恢复/救砖完成

9008模式

这是高通的soc提供的功能，只要没有硬件损坏，无论软件层面如何，都能启动该模式来操作/覆写整个flash

一般，棒子上都会提供一个复位按钮，在按住它的同时，对设备进行上电（上电后即可松开），这时应进入9008模式

注：对于410电池机，可能需要通过短接触点来进入9008模式，这些触点一般标注为FB

注注：如果短接触点不好使，试试拿镊子或刀片刮一刮它们，上面可能存在一层较厚的绝缘涂层

  • 进入9008模式后，所有指示灯不应被点亮，如果上电后发现指示灯被点亮，则代表没有成功进入9008模式，尝试下电重新操作
  • 每次使用9008后，建议下电并重新进入9008模式，即使是统一软件的不同功能，9008似乎在部分情况下不可在一次上电后被多次使用

9006模式

还有有一种类似的模式叫9006，在这个模式下，设备将作为一个块设备(硬盘)被识别，可以直接使用其他硬盘工具对设备进行分区(当u盘用),
但是注意，该模式依赖引导分区启动，如果在该模式下删除了所有分区，那么下一次下电重启就会因为啥也启动不了而进9008了

最后

刷过系统后，也可以进行全量备份，以便之后直接还原为自定义的系统
同一型号的备份包(甚至少部分不同型号的)可以通刷，但是大概率基带不通用导致4g异常（即使是统一型号的，也有概率出现该问题）。总之，建议为每一根棒子单独进行备份。
